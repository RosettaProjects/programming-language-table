# Boolean

## Python

Note

> Snippet:

```py
bool1 = True
bool2 = False
print(f"{bool1=}, {bool2=}")
```

> Output:

```txt
bool1=true, bool2=false
```

## Haskell

Note

> Snippet:

```hs
bool1 :: Bool
bool1 = True
 
bool2 :: Bool
bool2 = False
 
main :: IO ()
main = putStrLn $ "bool1=" ++ show bool1 ++ ", bool2=" ++ show bool2
```

> Output:

```txt
❯ bool1=true, bool2=false
```

## Rust

Note

> Snippet:

```rs
let bool1 = true;
let bool2 = false;
println!("bool1={bool1}, bool2={bool2}");
```

> Output:

```txt
❯ bool1=true, bool2=false
```

> Full:

```rs
fn main() {
    let bool1 = true;
    let bool2 = false;
    println!("bool1={bool1}, bool2={bool2}");
}
```
