# Haskell

## 0. Basics

### 0.0 Console Output

#### 0.0.0 Hello, world!

> Note

Snippet:

```hs
main :: IO ()
main = putStrLn "Hello, world!"
```

Full:

```hs
main :: IO ()
main = putStrLn "Hello, world!"
```

Output:

```txt
❯ Hello, world!
```

### 0.1. Primitive Types

#### 0.1.0 Boolean

> Note

Full:

```hs
bool1 :: Bool
bool1 = True
 
bool2 :: Bool
bool2 = False
 
main :: IO ()
main = putStrLn $ "bool1=" ++ show bool1 ++ ", bool2=" ++ show bool2
```

Output:

```txt
❯ bool1=true, bool2=false
```
